Read `_codeguide/runtime/navigation-issues.md`.

For each flagged issue, assess:
- What was missing or unclear in `_codeguide/` that caused the navigation failure?
- Was the threshold triggered because the guide was not consulted, or because the guide failed to route clearly?

Classify each proposal:
- **Local fix** → propose a change to `_codeguide/local-rules.md` or an Overview routing table
- **Upstream proposal** → flag as a generic guide improvement (not actionable in this repo — note it for the user to collect later)

Present proposals to the user before making any changes to `_codeguide/`.

Do not modify `_codeguide/` directly during this review.
