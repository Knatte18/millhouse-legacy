# Local Rules

Repo-specific additions to the Documentation Guide. These rules apply on top of
the generic guide and are not shared across repos.

Add rules here as you discover patterns specific to this codebase. Periodically
review local-rules files across repos to identify rules worth promoting to the
generic DocumentationGuide.md.
