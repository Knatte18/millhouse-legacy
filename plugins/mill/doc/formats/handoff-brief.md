# Handoff Brief — Deprecated

Handoff briefs are no longer used. `mill-start` derives the task slug from the branch name (see `millpy.core.paths.slug_from_branch`) and reads the task entry from `Home.md` in the wiki.

The `_millhouse/handoff.md` file is removed from the system. There is no replacement — the wiki's `Home.md` entry (description + Background link) provides the same context that the handoff brief previously carried.
