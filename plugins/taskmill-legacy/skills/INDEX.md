# Taskmill-legacy Skills

| Skill | Description |
|---|---|
| [legacy-mill-add](legacy-mill-add/SKILL.md) | Add a checkbox item to a file |
| [legacy-mill-add-discuss](legacy-mill-add-discuss/SKILL.md) | Add a task and start discussing it |
| [legacy-mill-commit](legacy-mill-commit/SKILL.md) | Commit and push (no rebase) |
| [legacy-mill-discuss](legacy-mill-discuss/SKILL.md) | Discuss a backlog task without writing a plan |
| [legacy-mill-do](legacy-mill-do/SKILL.md) | Implement the next planned task (does not commit) |
| [legacy-mill-do-all](legacy-mill-do-all/SKILL.md) | Implement all planned tasks, committing after each |
| [legacy-mill-do-commit](legacy-mill-do-commit/SKILL.md) | Implement the next planned task and commit |
| [legacy-mill-finalize](legacy-mill-finalize/SKILL.md) | Write a plan from the current discussion |
| [legacy-mill-finalize-do](legacy-mill-finalize-do/SKILL.md) | Finalize the current discussion and implement that task (no commit) |
| [legacy-mill-finalize-do-all](legacy-mill-finalize-do-all/SKILL.md) | Finalize, then implement all planned tasks |
| [legacy-mill-finalize-do-commit](legacy-mill-finalize-do-commit/SKILL.md) | Finalize, implement, and commit |
| [legacy-mill-formats](legacy-mill-formats/SKILL.md) | Backlog, changelog, and plan file format specs. Use when reading or writing task files. |
| [legacy-mill-list](legacy-mill-list/SKILL.md) | Show task status and pick one to discuss |
| [legacy-mill-log](legacy-mill-log/SKILL.md) | Generate a work-journal entry from recent commits |
| [legacy-mill-retry](legacy-mill-retry/SKILL.md) | Retry the first blocked task |
| [legacy-mill-review](legacy-mill-review/SKILL.md) | Review a plan by spawning a fresh agent with no conversation context |
